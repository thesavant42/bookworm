## TASK Port calibre-opds-client to a standalone cli utility

- [ code ](/code-to-port/calibre-opds-client/calibre_plugin/)
- This is meant to be a plugin to teh Calibre client, but we can leverage the useful patterns to enable a command line port of the tool, called "bookworm"

### MVP
- A Command line utility that accepts a calibre opds endpoint, such as `http://69.144.163.41:8080/opds` and enables the user to:
    - Search for keywords
    - browse libraries
    - Download titles ad hoc
    - Download titles in batched queues

#### Mockup:

```sh
python bookworm.py --target="http://69.144.163.41:8080/opds" # default to `browse`
python bookworm.py --target="http://69.144.163.41:8080/opds" --search="haunted mansion" # Results contain download links
                                                                                        # http://69.144.163.41:8080/#library_id=books&panel=book_list&search=haunted%20mansion&sort=timestamp.desc
python bookworm.py --target="http://69.144.163.41:8080/opds" --search="haunted mansion" --save-all --output-folder="." # save all results, regardless of format or name
python bookworm.py --target="http://69.144.163.41:8080/opds" --save="1774746720082"    # "1774746720082" is a book id
```
